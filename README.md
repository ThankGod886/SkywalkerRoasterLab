# SkywalkerRoasterLab
這是我修改的程式，讓 Skywalker Roaster 能夠使用USB線連接到 Artisan 和 無線藍芽連接到HiBean。它使用 ESP32-S3-DevKitC-1 作為控制器。
This is my modified program that allows the Skywalker Roaster to connect to Artisan via USB cable and to HiBean via Bluetooth. It uses the ESP32-S3-DevKitC-1 as the controller.
![image](https://github.com/ThankGod886/SkywalkerRoasterLab/blob/main/esp32s3devkitc1.jpg)

Develop a Modbus device using an Arduino Uno or Nano and a MAX6675 (KType) or MAX31865 (RTD). Connect it to a computer via USB to add a second temperature probe to the Skywalker. In Artisan, simply add an extra device to use it. The original ESP32 S3 controller can run simultaneously without any changes.
